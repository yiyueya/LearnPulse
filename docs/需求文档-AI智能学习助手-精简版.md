# AI 智能学习助手 - 需求文档（精简版）

> 版本：V1.0
> 日期：2026-04-20
> 定位：小学生（1-2年级，人教版）语数查漏补缺专用

---

## 1. 核心目标

帮助家长快速了解孩子**语数两科的薄弱点**，针对性地出题练习，直到掌握。

---

## 2. 用户

- 无需区分家长和孩子角色，使用同一用户
- 功能包括：查看薄弱点报告、推送练习、监督进度、做诊断测试、做练习题

---

## 3. 功能模块

### 3.1 课程内容管理

**数据来源**：本地PDF格式的人教版教材内容

**内容管理**：
- 从PDF文档中提取电子教材/课本内容（课文、知识点）
- 课程视频链接（如适用）
- 配套资源（课件、习题）

**知识地图**：
- AI自动提取PDF中的知识点
- 构建知识点之间的关联关系
- 自动生成知识体系结构
- 支持根据输入文档动态扩展范围

**目录结构**：
1. AI自动解析PDF文档结构
2. 按教材目录结构展示单元/章节
3. 点击单元 → 查看该单元的课程内容
4. 支持离线查看

**出题范围**：
- 基于知识地图的全范围诊断
- 可精确选择某个知识点进行专项测试
- 可选择多个知识点组合测试

**技术实现**：
- PDF文档解析与内容提取
- AI知识图谱构建
- 内容索引与检索
- 自动更新机制（基于新PDF文档）

---

### 3.2 学习诊断测试

**入口**：选择学科（语/数） → 选择知识地图中的知识点范围 → 开始诊断

**出题范围**：
- 全范围诊断：覆盖知识地图中的所有知识点
- 单/多知识点测试：精确选择1个或多个知识点
- 薄弱知识点专项：基于历史错题定位薄弱知识点

**测试流程**：
1. 基于知识地图，AI 生成 10-15 道题（约8分钟）
2. 根据选择的知识点动态生成题目
3. 题目根据知识点的重要性和关联度分布

**题目来源**：PDF提取内容 + AI 生成，难度分为：
- 基础题（70%）：课本原题变式
- 进阶题（25%）：综合运用
- 拓展题（5%）：稍有难度

**题目类型**：
- 选择题：4选项单选题
- 填空题：直接填写答案
- 判断题：对/错判断
- 应用题：简单应用题，只需答案

---

### 3.3 薄弱点分析报告

**输出形式**：

```
📊 数学 - 掌握度

已掌握 ████████░░ 80%
├─ 计算（100%）
├─ 认识图形（80%）
└─ 应用题（60%）⚠️

薄弱项：
1. 应用题 - 理解题意能力弱
2. 加减法 - 进位退位不熟练

建议：
- 加强应用题练习，重点训练关键词提取
- 每天练习5分钟加减法口算

📊 语文 - 掌握度

已掌握 ███████░░ 70%
├─ 拼音（85%）
├─ 汉字（70%）
└─ 句子（55%）⚠️

薄弱项：
1. 句子 - 造句能力弱
2. 汉字 - 笔顺不规范

建议：
- 每天练习2个句子仿写
- 加强汉字笔顺练习
```

**可视化**：
- 雷达图：展示各知识点掌握度
- 柱状图：对比不同知识点的掌握情况
- 趋势图：展示掌握度变化趋势

**报告周期**：
- 每次诊断后生成
- 每周自动生成周报
- 每月自动生成月报

---

### 3.4 专项练习

**按薄弱点出题**：
- 点击薄弱知识点 → 生成10道专项练习
- 练完自动判分
- 做错的题自动进入错题本
- 练习结果实时反馈

**练习模式**：
- 闯关模式：每天3关，每关10题，关卡难度递增
- 自由练习：随时可以做，支持自定义题量
- 限时挑战：在规定时间内完成题目，提升速度
- 知识点专项：针对特定知识点进行集中练习

**练习反馈**：
- 每道题即时显示正确答案和解析
- 练习完成后生成详细报告
- 推荐相关知识点的学习资源

---

### 3.5 错题本

**功能**：
- 自动收集所有做错的题
- 显示正确答案和详细解题思路
- 按知识点分类管理错题
- 支持手动添加错题
- 定期推送复习，直到做对为止

**复习机制**：
- 基于艾宾浩斯遗忘曲线安排复习时间
- 每道错题至少复习3次，间隔逐渐拉长
- 复习时可选择重新答题或查看解析
- 连续做对3次后从错题本中移除

**导出功能**：
- 支持导出错题为PDF或Word文档
- 支持打印错题集

---

### 3.6 学习进度追踪

**首页看板**：

```
📅 今日任务
✅ 数学加减法练习 5/10
⏳ 语文看图写话 0/1

📈 本周进步
数学：65分 → 78分 (+13)
语文：70分 → 72分 (+2)

🏆 学习成就
连续学习7天 ✅
完成100道题 ✅
掌握5个知识点 ✅
```

**详细统计**：
- 总学习时长
- 完成题目数量
- 正确率变化趋势
- 语数两科学习时间分布

**目标设置**：
- 可设置每日学习目标
- 可设置每周进步目标
- 完成目标后获得成就奖励

---

### 3.7 复习提醒

**提醒类型**：
- 错题复习提醒：按艾宾浩斯遗忘曲线推送
- 每日练习提醒：固定时间提醒做练习
- 学习目标提醒：提醒完成每日/每周目标
- 薄弱点强化提醒：针对未掌握的知识点

**提醒设置**：
- 家长可设置提醒时间和频率
- 支持多种提醒方式（应用内、微信、短信）
- 可根据孩子的学习习惯调整提醒策略

**智能提醒**：
- 基于学习进度自动调整提醒频率
- 避免在孩子休息时间发送提醒
- 提醒内容个性化，包含具体学习任务

---

### 3.8 多AI Agent架构

**架构设计**：
- 基于Agent的分布式智能系统
- 各个Agent负责不同的功能模块
- 统一由调度中心协调工作

**核心Agent**：
1. **内容提取Agent**：
   - 负责从PDF文档中提取知识点
   - 使用阿里云百炼多模态模型处理PDF内容
   - 生成结构化的知识点数据

2. **知识图谱Agent**：
   - 构建和维护知识地图
   - 分析知识点之间的关联关系
   - 优化知识体系结构

3. **出题Agent**：
   - 基于知识点生成题目
   - 确保题目符合年级难度
   - 多样化题目类型

4. **判题Agent**：
   - 自动判断答案正确性
   - 提供详细的解析
   - 识别常见错误类型

5. **分析Agent**：
   - 分析学习数据
   - 生成薄弱点报告
   - 提供个性化学习建议

**协作流程**：
1. 内容提取Agent处理PDF文档
2. 知识图谱Agent构建知识地图
3. 出题Agent基于知识地图生成题目
4. 判题Agent评估答题结果
5. 分析Agent生成学习报告

**技术实现**：
- 使用阿里云百炼多模态模型作为核心AI能力
- Agent间通过消息队列通信
- 统一的任务调度系统

## 5. 技术方案（简化）

| 模块 | 技术 |
|-----|-----
| 前端 | Vue.js + Tailwind CSS |

## 7. 备注

- 无需用户系统，家长和孩子使用同一界面
- 题库先用 JSON 文件管理，后续可扩展数据库
- 课程内容使用本地PDF格式的人教版教材
- 采用多AI Agent架构，使用阿里云百炼多模态模型处理数据输入
- AI自动从PDF中提取知识点并构建知识地图
- 支持通过上传新PDF文档动态扩展知识范围
- 题目生成、判题等功能均由AI Agent完成
- 各Agent间通过消息队列通信，统一由调度中心协调
